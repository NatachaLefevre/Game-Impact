# Game-Impact

Créations visuelles, mises en page et adaptations de charte graphique sur différents supports (flyers, livrets, cartes de visite, habillage streaming...).

Game Impact est une association qui promeut une vision du jeu vidéo comme acteur de futurs souhaitables, qui agit pour la transition du jeu vidéo vers plus de sens.

https://gameimpact.fr/
